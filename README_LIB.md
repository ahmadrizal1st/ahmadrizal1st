# Development Concepts Overview

Berikut adalah ikhtisar jenis-jenis komponen pengembangan perangkat lunak yang umum digunakan.

## Starter Kits

| Name | Description | Name | Description |
| :--- | :--- | :--- | :--- |
| **[Create React App](https://create-react-app.dev/)** | Starter kit resmi untuk mempelajari dan membangun aplikasi web modern dengan React. Menangani semua setup tooling seperti Webpack dan Babel. | **[Vite](https://vitejs.dev/)** | Generator project yang sangat cepat dan minimalist untuk berbagai framework (Vue, React, Svelte, dll). Menawarkan Developer Experience yang unggul. |
| **[Next.js Starter](https://nextjs.org/docs/api-reference/create-next-app)** | Template untuk memulai project React dengan framework Next.js yang memiliki fitur seperti SSR, SSG, dan routing berbasis file. | **[Nuxt.js Starter](https://nuxt.com/docs/getting-started/introduction)** | Starter kit untuk aplikasi Vue.js dengan framework Nuxt.js, menyediakan konvensi struktur project dan optimasi yang kuat. |
| **[Angular CLI](https://angular.io/cli)** | Command Line Interface untuk generate project Angular lengkap dengan konfigurasi testing, routing, dan build optimizer yang sudah siap. | **[Create T3 App](https://create.t3.gg/)** | Starter kit opiniated untuk membangun aplikasi full-stack typesafe dengan tRPC, Next.js, Prisma, dan Tailwind CSS. |
| **[Expo CLI](https://docs.expo.dev/workflow/expo-cli/)** | Tool untuk memulai project React Native tanpa perlu konfigurasi native code yang rumit, memungkinkan development yang sangat cepat. | **[Laravel Jetstream](https://jetstream.laravel.com/)** | Starter kit untuk aplikasi Laravel yang menyediakan scaffolding untuk login, registrasi, dan fitur tim dengan Livewire atau Inertia. |

## Frameworks

| Name | Description | Name | Description |
| :--- | :--- | :--- | :--- |
| **[React](https://reactjs.org/)** | Library JavaScript untuk membangun antarmuka pengguna (UI) yang bersifat deklaratif dan berbasis komponen. | **[Vue.js](https://vuejs.org/)** | Framework JavaScript progresif untuk membangun UI yang approachable, versatile, dan performant. |
| **[Angular](https://angular.io/)** | Platform dan framework untuk membangun aplikasi web single-page client-side yang komprehensif dan menggunakan TypeScript. | **[Svelte](https://svelte.dev/)** | Framework JavaScript yang menggeser pekerjaan dari runtime ke step compile untuk menghasilkan kode vanilla JS yang sangat efisien. |
| **[Spring Boot](https://spring.io/projects/spring-boot)** | Framework Java yang menyederhanakan pembuatan aplikasi stand-alone, production-grade berbasis Spring. | **[Express.js](https://expressjs.com/)** | Framework web untuk Node.js yang minimal, fleksibel, dan menyediakan fitur yang kuat untuk aplikasi web dan API. |

## Libraries

| Name | Description | Name | Description |
| :--- | :--- | :--- | :--- |
| **[Axios](https://axios-http.com/)** | Library client HTTP berbasis Promise untuk browser dan Node.js untuk membuat request HTTP. | **[Lodash](https://lodash.com/)** | Library utilitas JavaScript yang menyediakan fungsi untuk mempermudah manipulasi array, object, dan lainnya. |
| **[Redux](https://redux.js.org/)** | Library untuk mengelola state aplikasi yang dapat diprediksi dalam aplikasi JavaScript. | **[jQuery](https://jquery.com/)** | Library JavaScript yang cepat dan kaya fitur yang menyederhanakan traversing DOM, event handling, dan animasi. |

## SDKs (Software Development Kits)

| Name | Description | Name | Description |
| :--- | :--- | :--- | :--- |
| **[Android SDK](https://developer.android.com/studio)** | Kumpulan tools, library, dan dokumentasi untuk mengembangkan aplikasi pada platform Android. | **[iOS SDK](https://developer.apple.com/ios/)** | Kumpulan tools dan framework dari Apple untuk mengembangkan aplikasi untuk perangkat iOS seperti iPhone dan iPad. |
| **[AWS SDK](https://aws.amazon.com/sdk-for-javascript/)** | Kumpulan library yang memungkinkan developer untuk berinteraksi dengan layanan Amazon Web Services (AWS) dari kode mereka. | **[Stripe SDK](https://stripe.com/docs/development)** | Library untuk mengintegrasikan pembayaran, checkout, dan operasi bisnis lainnya dari Stripe ke dalam aplikasi. |

## Tools / CLIs

| Name | Description | Name | Description |
| :--- | :--- | :--- | :--- |
| **[Webpack](https://webpack.js.org/)** | Static module bundler untuk aplikasi JavaScript modern yang memaketkan modul-modul kode menjadi satu atau beberapa bundle. | **[ESLint](https://eslint.org/)** | Tool untuk mengidentifikasi dan melaporkan pola dalam kode JavaScript, membantu membuat kode tetap konsisten dan bebas bug. |
| **[Prettier](https://prettier.io/)** | Opinionated code formatter yang mendukung banyak bahasa untuk memastikan style code yang konsisten. | **[Docker CLI](https://docs.docker.com/engine/reference/commandline/cli/)** | Command Line Interface untuk berinteraksi dengan Docker daemon, digunakan untuk mengelola container, image, dan jaringan. |
